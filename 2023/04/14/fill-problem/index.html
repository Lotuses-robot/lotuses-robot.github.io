初赛自创填空题

  • 2023-04-14
  • 作者 Lotuses
  • ~5.17K 字

实验室中有一个由 nnn 个实验腔连接而成的环形轨道，其中第 iii 个实验腔顺时针连向第 i+1i+1i+1 个实验腔（特别的，第 nnn 个实验腔连向第 111 个实验腔）。同时还有一个标号为 n+1n+1n+1 的新建实验腔要接入这个环形轨道。它可以接在任意两个原本相连的实验腔之间。

第 iii 个实验腔可以将带电荷量为 QQQ 的粒子运输到它的下一个实验腔，这个过程花费的能量为 ∣Q∣×ci\vert Q \vert \times c_i∣Q∣×ci​。除此之外，第 iii 个实验腔本身就存储了量为 eie_iei​ 的电荷（电荷量有正负）。由于众所周知的电荷守恒定律，第 n+1n+1n+1 个实验腔储存的电荷量与前 nnn 个实验腔储存的总电荷量的代数和为 000。

小 Z 有一个原本不带电的粒子。等到第 n+1n+1n+1 个实验腔接入轨道后，他要任选一个实验腔（包括第 n+1n+1n+1 个）作为出发点，将粒子放入，并使之在实验腔的能量驱动下顺时针环游一周回到出发点。粒子每到达一个实验腔（包括出发点），它所带电荷量就会变成原来所带的电荷量和这个实验腔所储存的电荷量的代数和。

注意：电荷量会先加上实验腔所含电荷量，再计算能量贡献。

现在，小 Z 想知道，在所有接入新建实验腔并选定出发点的方案中，粒子环游一周所需的能量最少为多少？

提示：

∑j=1n+1∣sb′−sj′∣⋅cj\sum_{j = 1} ^ {n + 1} |s_b' - s_j'| \cdot c_j j=1∑n+1​∣sb′​−sj′​∣⋅cj​

本代码做法是维护一个 ∑s⋅c\sum s \cdot c∑s⋅c 和 ∑c\sum c∑c 之后用二分寻找分界点 cic_ici​ 后直接统计。省略了读入和输出函数 read 和 writeln。

  1. A. c * s B. c * s * pos C. s D. s * pos

  2. A. nds[nds[u].lc].c <= cpos B. nds[nds[u].rc].c <= cpos

    C. c + nds[nds[u].lc].c <= cpos D. c + nds[nds[u].rc].c <= cpos

  3. A. (lc - rc) * qs + rx - lx B. (lc - rc) * qs + lx - rx

    C. (rc - lc) * qs + rx - lx B. (rc - lc) * qs + lx - rx

  4. A. s[i] + e[n + 1] B. s[i] - e[n + 1] C. s[i - 1] + e[n + 1] D. s[i - 1] - e[n + 1]

  5. A. n * 2 + 1 B. n - 1 C. n D. n + 1
